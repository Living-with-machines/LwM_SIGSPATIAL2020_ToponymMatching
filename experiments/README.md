# README

This folder contains two notebooks with the [experiments](https://github.com/Living-with-machines/LwM_SIGSPATIAL2020_ToponymMatching/tree/master/experiments) reported in the paper:
* [Toponym_Matching_Experiments.ipynb](https://github.com/Living-with-machines/LwM_SIGSPATIAL2020_ToponymMatching/blob/develop/experiments/Toponym_Matching_Experiments.ipynb) has the experiments summarized in table 1 of the SIGSPATIAL article.
* [Candidate_Ranking_Experiments.ipynb](https://github.com/Living-with-machines/LwM_SIGSPATIAL2020_ToponymMatching/blob/tm_experiments/experiments/Candidate_Ranking_Experiments.ipynb) has the experiments summarized in table 2 of the SIGSPATIAL article. 

:warning: Make sure you have gone through the required `processing` steps (described [here](https://github.com/Living-with-machines/LwM_SIGSPATIAL2020_ToponymMatching/tree/master/processing)) and that you have all data needed before you run the experiments.
